# AI Chat Assistant (Week 1)

This is a simple AI Chat Assistant built as part of the DStarix genai Internship. It uses the Google Gemini API to answer user queries through a Web/CLI interface.

## 🚀 Features
- Powered by Gemini LLM (`gemini-2.5-flash`).
- Interactive UI using Streamlit (or CLI).
- Secure API key management using `.env`.
